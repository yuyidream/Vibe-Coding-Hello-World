## 🛠️ 部署工具说明

### 1. deploy.sh（服务器端脚本）

**位置：** 需要上传到ECS服务器

**功能：**
- 自动检测Linux发行版（Ubuntu/CentOS）
- 安装Nginx
- 创建网站目录
- 配置Nginx虚拟主机
- 启动并设置开机自启

**使用方法：**
```bash
chmod +x deploy.sh
bash deploy.sh
```

### 2. 快速部署.bat（Windows客户端工具）

**位置：** 在本地Windows电脑上运行

**功能：**
- 检查本地文件完整性
- 测试SSH连接
- 自动上传文件到服务器
- 自动执行部署脚本
- 智能错误提示

**使用方法：**
```powershell
.\快速部署.bat 你的公网IP
```

### 3. nginx-site.conf（配置文件模板）

**用途：** Nginx配置参考

**包含特性：**
- 基础HTTP服务配置
- 静态文件缓存优化
- 安全头部设置
- HTTPS配置模板（注释）

## 🔐 安全建议

### 部署后必做

1. **修改SSH端口**
   - 默认22端口易被扫描
   - 建议改为高位端口（如2222）

2. **配置防火墙**
   ```bash
   apt install ufw -y
   ufw allow 22/tcp
   ufw allow 80/tcp
   ufw enable
   ```

3. **定期更新系统**
   ```bash
   apt update && apt upgrade -y  # Ubuntu
   yum update -y                 # CentOS
   ```

### 增强安全（可选）

- 使用SSH密钥认证代替密码
- 安装Fail2ban防暴力破解
- 启用HTTPS（需要SSL证书）
- 配置日志监控

## 🌐 后续扩展

### 绑定域名

**前提：** 已有域名

**步骤：**
1. 在域名提供商添加A记录
2. 指向ECS公网IP
3. 修改Nginx配置中的`server_name`
4. 重载Nginx配置

### 启用HTTPS

**使用Let's Encrypt免费证书：**

```bash
# 安装Certbot
apt install certbot python3-certbot-nginx -y

# 自动配置证书
certbot --nginx -d your-domain.com

# 测试自动续期
certbot renew --dry-run
```

### 添加更多页面

1. 创建新的HTML文件
2. 上传到 `/var/www/hello-world/`
3. 访问 `http://你的IP/新页面.html`

## 📊 性能优化

### Nginx优化

```nginx
# 开启gzip压缩
gzip on;
gzip_types text/plain text/css application/json application/javascript;

# 连接优化
keepalive_timeout 65;

```

## 💬 获取帮助

**遇到问题？**

1. 查看 [`部署操作指南.md`](部署操作指南.md) 的"故障排查"章节
2. 检查华为云ECS实例状态
3. 查看Nginx日志：`tail -f /var/log/nginx/hello-world-error.log`

**常见问题：**
- SSH连接失败 → 检查安全组配置
- 网站无法访问 → 检查Nginx服务状态
- 403错误 → 检查文件权限
