## 🎯 接下来由您完成的步骤

以下操作需要您在华为云控制台和本地电脑上完成：

### 步骤1：配置华为云安全组 ⏱️ 约5分钟

**需要您做的：**
1. 登录华为云控制台
2. 进入ECS管理页面
3. 配置安全组，开放端口：
   - 端口22（SSH）
   - 端口80（HTTP）
4. 记录您的ECS公网IP地址

**详细指南：** 请打开 [`华为云安全组配置指南.md`](华为云安全组配置指南.md)

**完成后请记录：**
```
✏️ 公网IP：_________________
✏️ 登录密码：_________________
```

---

### 步骤2：运行一键部署脚本 ⏱️ 约3分钟

**需要您做的：**

1. 打开PowerShell（Win+X → Windows PowerShell）

2. 进入项目目录：
   ```powershell
   cd "C:\编程工具\cursor的项目\Vibe-Coding-Hello-World"
   ```

3. 运行部署脚本：
   ```powershell
   .\快速部署.bat 你的公网IP
   ```

   **例如：**
   ```powershell
   .\快速部署.bat 123.45.67.89
   ```

4. 根据提示输入root密码

**脚本将自动完成：**
- ✅ 上传网站文件到服务器
- ✅ 上传部署脚本到服务器
- ✅ 在服务器上安装Nginx
- ✅ 配置Nginx
- ✅ 启动Nginx服务

---

## 🛠️ 如果遇到问题

### 问题1：快速部署脚本无法运行

**方案A：** 使用手动部署方式

详细步骤请查看：[`部署操作指南.md`](部署操作指南.md)

**方案B：** 检查PowerShell执行策略

```powershell
# 查看当前策略
Get-ExecutionPolicy

# 如果是Restricted，需要临时允许
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

### 问题2：SSH连接失败

**检查清单：**
- [ ] 安全组是否已开放22端口？
- [ ] IP地址是否正确？
- [ ] ECS实例是否正在运行？
- [ ] 密码是否正确？

**解决方案：**
- 返回华为云控制台重新检查配置
- 在ECS控制台重置密码
- 确认实例状态为"运行中"

### 问题3：网站无法访问

**检查清单：**
- [ ] 安全组是否已开放80端口？
- [ ] Nginx服务是否正常运行？
- [ ] 浏览器地址是否正确（http://而非https://）？

**验证Nginx状态：**
```bash
# SSH登录服务器后执行
systemctl status nginx
```

看到 `active (running)` 表示正常。

**重启Nginx：**
```bash
systemctl restart nginx
```
