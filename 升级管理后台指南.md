## 📝 部署步骤（详细）

### 1. 构建新版管理后台

```powershell
cd frontend-admin
npm install
npm run build
```

### 2. 上传到服务器

```powershell
# 回到项目根目录
cd ..

# 上传构建产物
scp -r frontend-admin/dist/* root@123.249.68.162:/www/wwwroot/hello-world/frontend-admin/
```

### 3. 更新 Nginx 配置

在服务器上修改 Nginx 配置，添加：

```nginx
# 新版管理后台（Vben Admin）
location /Vibe-Coding-Hello-World/admin {
    alias /www/wwwroot/hello-world/frontend-admin;
    index index.html;
    try_files $uri $uri/ /Vibe-Coding-Hello-World/admin/index.html;
}

# 旧版管理后台（保留）
location /Vibe-Coding-Hello-World/admin.html {
    alias /www/wwwroot/hello-world/admin.html;
}
```

### 4. 重启 Nginx

```bash
sudo systemctl reload nginx
```

### 5. 访问新版管理后台

```
https://renxinayi.com/Vibe-Coding-Hello-World/admin/
```

## 🔧 自定义和扩展

### 添加新页面

1. 在 `src/views/` 下创建新目录
2. 在 `src/router/index.ts` 中添加路由
3. 在 `src/layouts/BasicLayout.vue` 中添加菜单项
4. 在 `src/locales/lang/` 中添加国际化文本

### 修改主题

修改 `src/styles/index.scss` 或使用 Ant Design Vue 的主题配置。

### 添加新语言

1. 在 `src/locales/lang/` 下创建新语言文件
2. 在 `src/locales/index.ts` 中注册
