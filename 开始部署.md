# 🚀 开始部署 - 操作清单

欢迎！所有部署文件已准备完毕，请按照以下步骤完成部署。

---

## ✅ 部署前检查清单

在开始之前，请确认：

- [ ] 已购买华为云ECS服务器
- [ ] 已购买并绑定公网IP
- [ ] 知道ECS的登录密码
- [ ] 电脑已安装SSH客户端（Windows 10/11自带）

---

## 📋 三步完成部署

### 第一步：配置华为云安全组 ⏱️ 约5分钟

**请打开文件：** `华为云安全组配置指南.md`

按照指南完成以下操作：
1. 登录华为云控制台
2. 找到您的ECS实例
3. 开放22端口（SSH）
4. 开放80端口（HTTP）
5. 记录公网IP地址

**完成后，请记录：**
```
公网IP：_________________
密码：  _________________
```

---

### 第二步：一键部署 ⏱️ 约3分钟

打开PowerShell（Win+X → Windows PowerShell），执行：

```powershell
cd "C:\编程工具\cursor的项目\Vibe-Coding-Hello-World"
.\快速部署.bat 你的公网IP
```

**例如：**
```powershell
.\快速部署.bat 123.45.67.89
```

脚本将自动：
- ✅ 上传文件到服务器
- ✅ 安装Nginx
- ✅ 配置网站
- ✅ 启动服务

---

### 第三步：访问网站 ⏱️ 立即完成

在浏览器中打开：
```
http://你的公网IP
```

看到Hello World页面即表示部署成功！🎉

---

## 🛠️ 遇到问题？

### 问题1：快速部署失败

**解决方案：** 使用手动部署方式

打开文件：`部署操作指南.md`，按照详细步骤操作。

### 问题2：SSH连接失败

**可能原因：**
- 安全组未正确配置
- IP地址输入错误
- 密码错误

**解决方案：**
1. 重新检查安全组配置
2. 在华为云控制台验证IP地址
3. 重置ECS密码（如果需要）

### 问题3：浏览器无法访问

**检查步骤：**
1. 安全组是否开放80端口？
2. Nginx服务是否运行？（SSH登录后执行：`systemctl status nginx`）
3. 防火墙是否拦截？

详细故障排查请参考：`部署操作指南.md` 中的"故障排查"章节

## 🎯 部署后的下一步

### 管理网站

**更新网站内容：**
```powershell
# 修改index.html后，重新上传
scp index.html root@你的公网IP:/var/www/hello-world/
```

**重启Nginx：**
```bash
# SSH登录后执行
systemctl restart nginx
```

### 安全加固

部署成功后，建议：
1. 修改SSH默认端口
2. 配置防火墙
3. 定期更新系统
4. 启用HTTPS（如果有域名）

详见：`部署操作指南.md` 的"安全建议"章节

### 绑定域名（可选）

如果您有域名，可以：
1. 添加DNS A记录指向公网IP
2. 修改Nginx配置添加域名
3. 申请SSL证书启用HTTPS

详见：`部署操作指南.md` 的"后续扩展"章节
