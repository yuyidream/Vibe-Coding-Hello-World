- README文件（通常命名为README.md）是项目的说明文档，用于向开发者介绍项目。它通常包含项目概述、安装步骤、使用说明、贡献指南等信息。

主要特点：

介绍性：向开发者介绍项目是什么、如何运行、如何贡献等。

通用性：几乎所有开源项目都有一个README文件，用于提供项目的基本信息。

面向开发者：主要供阅读和了解项目的人使用。

- 仓库门面，包含：简介、核心特性、快速开始、其它重要文件的说明和索引
