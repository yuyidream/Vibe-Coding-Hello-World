# 华为云安全组配置指南

## 第一步：配置安全组规则

在开始部署之前，您需要在华为云控制台配置安全组规则，开放必要的端口。

### 操作步骤

1. **登录华为云控制台**
   - 访问：https://console.huaweicloud.com
   - 使用您的账号登录

2. **进入ECS管理界面**
   - 在顶部导航栏找到"控制台"
   - 在左侧菜单找到"弹性云服务器 ECS"
   - 点击进入ECS管理页面

3. **找到您的服务器**
   - 在实例列表中找到您购买的ECS服务器
   - 记录下服务器的**公网IP地址**（非常重要！）

4. **配置安全组**
   - 点击您的ECS实例名称
   - 在实例详情页面，找到"安全组"选项卡
   - 点击安全组名称进入安全组详情

5. **添加入方向规则**
   
   点击"入方向规则" → "添加规则"，添加以下两条规则：

   **规则1：SSH端口（用于远程连接）**
   - 协议：TCP
   - 端口：22
   - 源地址：0.0.0.0/0
   - 描述：SSH远程连接

   **规则2：HTTP端口（用于网站访问）**
   - 协议：TCP
   - 端口：80
   - 源地址：0.0.0.0/0
   - 描述：HTTP网站访问

6. **保存配置**
   - 点击"确定"保存规则
   - 等待规则生效（通常立即生效）

## 验证配置

配置完成后，您可以先测试SSH连接：

### Windows系统

打开PowerShell或命令提示符，输入：
```powershell
ssh root@你的公网IP
```

### 如果没有SSH客户端

Windows 10/11通常自带SSH，如果提示命令不存在，可以：
- 使用PuTTY工具（下载地址：https://www.putty.org/）
- 或在Windows设置中启用OpenSSH客户端
