# 数据库配置说明

## ✅ 当前配置（华为云 RDS）

### 数据库信息

- **云服务商**: 华为云
- **产品**: RDS for MySQL
- **版本**: MySQL 5.7.38-230100-log
- **连接方式**: VPC 内网连接
- **内网IP**: 192.168.0.243
- **端口**: 3306
- **数据库名**: hello_world

### 网络拓扑

```
华为云 VPC (192.168.0.0/24)
├── ECS 服务器: 192.168.0.215
└── RDS 数据库: 192.168.0.243

        ├── site_config (配置)

```

### 应用配置

**环境变量** (`.env`):
```bash
MYSQL_HOST=192.168.0.243
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=YU@yi811231
MYSQL_DATABASE=hello_world
```

### 验证连接

```bash
# 在服务器上测试连接
mysql -h 192.168.0.243 -u root -p -D hello_world

# 查看版本
SELECT VERSION();
# 输出: 5.7.38-230100-log

# 查看表
SHOW TABLES;
# 输出:
# - admin_users
# - site_config
# - access_logs
```
